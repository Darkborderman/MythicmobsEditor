# MythicmobsEditor

Mythicmobs GUI editor created by PyQt.

Inspired by lots of long, complicated and hard-to-maintain Mythicmobs YML config files.

## Prerequisitive

* Qt5 framework SDK
* pipenv (python environment manager)

## Development

1. Initialize and enter the environment
```
make init
pipenv shell
```

2. Execute the application
```
python3 editor/app.py
```

2. Linting
```
make lint
```

## Deployment

```
make bundle
```
